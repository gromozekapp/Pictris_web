Frequently Asked Questions

Quick answers about PicTris, Blogger Mode, and the built-in teleprompter.

How do I create a text file for the teleprompter on iPhone?

In Blogger Mode, the teleprompter loads plain .txt files from the Files app. Here is the easiest way to create one:

  1. Open Notes
  2. Tap the pencil icon in the top-right corner to start a new note
  3. Write your script
  4. Tap Share in the top-right corner → Save to Files
  5. Choose a folder and add .txt to the end of the filename (for example, my-script.txt)
  6. Tap Save

In PicTris, open Blogger Mode → teleprompter → load the file from Files.

How do I load a script into the teleprompter?
  1. Start a game in Blogger Mode
  2. Open the teleprompter panel
  3. Tap to import a .txt file from the Files app
  4. Choose auto scroll or manual swipe to read while you play
Is PicTris free?

Yes. The core game — Classic, Gallery, and basic Blogger Mode — is free with no ads. PicTris Pro is an optional In-App Purchase that unlocks unlimited Blogger Mode recordings, no watermark on exports, and the full teleprompter.

Does PicTris work offline?

Yes. Core gameplay works completely offline. Your photos, videos, and recordings stay on your device — nothing is uploaded to the cloud.

Why is there no sound in my Blogger Mode recording?

Make sure PicTris has Microphone permission in iOS Settings → PicTris. Screen recording captures gameplay video; the microphone records your voice separately during export.
